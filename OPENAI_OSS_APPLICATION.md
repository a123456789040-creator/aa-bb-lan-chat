## Brief description of the project

AA BB LAN Chat is a lightweight open-source LAN chat application built with Node.js, Express, and Socket.IO. It is designed for simple real-time communication between three fixed participants, `AA`, `BB`, and `CC`, on the same trusted network. The project is intentionally small and easy to inspect, making it useful as a reference implementation for local collaboration tools, realtime Socket.IO workflows, and lightweight team coordination prototypes.
