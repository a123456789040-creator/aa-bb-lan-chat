# AA BB LAN Chat v1.0.0

## Highlights

- Realtime LAN chat for three fixed roles: `AA`, `BB`, and `CC`
- Browser-based UI with presence indicators and typing updates
- LAN URL discovery for quick testing across multiple devices
- Helper scripts for smoke tests, history reads, scripted messaging, and demo seeding
- Public project docs, issue templates, CODEOWNERS, and CI

## Included in this release

- Core Express and Socket.IO server
- Static frontend UI
- Smoke test coverage
- Health check endpoint
- Demo screenshot and README walkthrough
- Maintainer-oriented repository metadata and contribution docs

## Recommended next steps

- Add persistent message storage
- Add configurable room and role presets
- Add lightweight access controls for semi-shared LAN environments
